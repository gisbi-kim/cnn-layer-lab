CNN 기본 예시: Conv 2개짜리 작은 네트워크

여기서 “2-layer CNN”이라고 부르면 보통 convolution 레이어가 2개라는 뜻입니다. 위 실험 옵션에서는 Conv 레이어 수를 1개부터 5개까지 바꿔 비교할 수 있습니다.

Input 28x28x1
흑백 숫자
->
Conv 1 3x3 filter 8개
28x28x8
->
MaxPool 1 2x2로 줄임
14x14x8
->
Conv 2 3x3 filter 16개
14x14x16
->
MaxPool 2 2x2로 줄임
7x7x16
->
Flatten + Dense 784개 숫자
32개 판단 특징
->
Softmax 0-9 확률
10개 출력
Convolution 작은 필터가 이미지를 훑으며 선, 모서리, 획 같은 특징을 찾습니다.
Feature map 필터가 어디에 강하게 반응했는지 그린 중간 결과입니다.
Pooling 가장 강한 반응만 남겨 크기를 줄이고 위치 변화에 덜 민감하게 만듭니다.
Dense 앞에서 찾은 특징들을 조합해 숫자 0-9 중 무엇인지 판단합니다.
